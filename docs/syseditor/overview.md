# 组态编辑器概述

## 什么是组态编辑器

组态编辑器是一种用于创建、编辑和管理组态画面的可视化工具，它允许用户通过拖拽、配置等方式快速构建各种工业监控、数据可视化等场景。

## 主要功能

- **可视化编辑**：通过拖拽方式快速创建和编辑组态画面
- **数据可视化**：实时展示设备数据，支持各种图表和仪表盘
- **脚本支持**：支持 JavaScript 脚本，实现复杂的逻辑和交互
- **多平台支持**：支持 Web 浏览器访问，无需安装客户端

## 应用场景

- 工业监控系统
- 数据中心可视化
- 能源管理系统
- 智能建筑监控
- 交通管理系统

## 技术特点

- 基于 HTML5 和 Canvas 技术
- 支持实时数据更新
- 响应式设计，适配不同屏幕尺寸
- 支持导出和导入组态画面

## 系统要求

- 现代 Web 浏览器（Chrome、Firefox、Safari、Edge）
- 支持 HTML5 和 Canvas
- 稳定的网络连接
- 配合 Opendlp 平台使用。
